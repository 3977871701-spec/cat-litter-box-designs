# 猫砂盆3D设计方案

面向越南市场的猫砂盆产品3D渲染方案。使用 Python + Pillow 生成等轴测（Isometric）风格的产品效果图，包含三款不同设计的猫砂盆，经过多个版本迭代优化，最终产出可用于电商展示的产品图片。

## 功能特性

- **三款产品设计**：
  - **型号A** — 三层顶入式防溅猫砂盆（80×60×55cm）：三层结构、顶部入口、防溅围边、侧面通风口
  - **型号B** — 单门翻盖式猫砂盆（60×50×50cm）：翻盖铰链设计、透明天窗、前开门、猫爪印花装饰
  - **型号C** — 抽屉式简约猫砂盆（55×45×45cm）：前拉抽屉、极简线条、猫砂指示线
- **等轴测3D渲染**：使用数学投影计算实现伪3D效果，包含顶面、正面、侧面三可视面
- **场景化展示**：渐变背景、地板纹理、绿植装饰、地毯等生活化场景元素
- **投影阴影**：高斯模糊软阴影，增强立体感
- **多版本迭代**：v1 基础版 → v2 增强阴影版 → v3 越南电商风格版
- **越南市场适配**：包含越南语标签、品牌标识（CatOi Vietnam/CaiOi Vietnam）
- **多配色方案**：莫兰迪色系（薄荷绿、珊瑚粉、薰衣草紫）+ 马卡龙色系

## 技术栈

| 组件 | 技术 |
|------|------|
| 图像生成 | Python + Pillow (PIL) |
| 数学计算 | NumPy |
| 3D投影 | 等轴测投影（30° 角度，cos=0.866, sin=0.5） |
| 输出格式 | PNG（1200×900px） |

## 安装方法

### 环境要求

- Python 3.8+
- macOS / Windows / Linux

### 安装步骤

```bash
# 安装依赖
pip install Pillow numpy
```

## 使用方法

### 生成效果图

```bash
# 运行 v1 基础版（输出到 ~/cat_litter_designs/）
python generate_litter_boxes.py

# 运行 v2 增强版（更好的阴影和细节）
python generate_litter_boxes_v2.py

# 运行 v3 越南电商风格版（输出到脚本同目录）
python generate_v3.py
```

### 输出说明

每个脚本运行后会生成三张 1200×900 像素的 PNG 效果图，分别对应三款猫砂盆设计。

| 版本 | 风格特点 | 输出文件 |
|------|----------|----------|
| v1 | 基础等轴测渲染 | `model_1/2/3.png` |
| v2 | 增强阴影 + 高斯模糊 + 边缘高光 | `v2_model_1/2/3.png` |
| v3 | 越南电商参考图风格 + 越南语标签 | `v3_model_a/b/c.png` |

### 配色方案

| 型号 | v1/v2 配色 | v3 配色 |
|------|-----------|---------|
| A | 莫兰迪薄荷绿 | 灰米色 + 深灰 |
| B | 马卡龙珊瑚粉 | 白色 + 浅灰 |
| C | 莫兰迪薰衣草紫 | 奶白 + 浅粉 |

### STL 文件

项目同时包含三款设计的 STL 格式3D模型文件，可用于：
- 3D打印打样
- 导入 Blender/Fusion 360 等软件进一步编辑
- 电商平台3D预览
